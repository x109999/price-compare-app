# 1688 开放平台账号申请指南

## 📋 申请流程

### 第一步：注册阿里巴巴账号

1. **访问注册页面**
   - 网址：https://login.1688.com/
   - 点击"免费注册"

2. **选择账号类型**
   - 个人账号：身份证注册
   - 企业账号：营业执照注册（推荐，权限更高）

3. **完成实名认证**
   - 个人：上传身份证 + 人脸识别
   - 企业：上传营业执照 + 法人认证

---

### 第二步：入驻开放平台

1. **访问开发者控制台**
   - 网址：https://open.1688.com/
   - 点击右上角"登录"

2. **创建应用**
   - 进入"控制台" → "我的应用"
   - 点击"创建新应用"

3. **填写应用信息**
   ```
   应用名称：即时零售比价工具
   应用类型：网站应用
   应用描述：帮助用户对比 1688 商品价格，提供比价服务
   回调地址：http://localhost:3000/callback（测试）
            https://yourdomain.com/callback（生产）
   ```

4. **选择 API 权限**
   - ✅ `user_info` - 获取用户信息（登录必需）
   - ✅ `order_info` - 获取订单信息（比价必需）
   - ✅ `item_search` - 搜索商品（可选）

5. **提交审核**
   - 审核时间：1-3 个工作日
   - 审核结果会短信通知

---

### 第三步：获取密钥

审核通过后，在应用详情页获取：

```
App Key: 你的 app_key（24 位数字）
App Secret: 你的 app_secret（32 位字母数字）
```

---

## ⚠️ 注意事项

### 1. 回调地址要求
- **测试环境**：`http://localhost:3000/callback`
- **生产环境**：必须是 `https://` 开头的域名
- 域名需要 ICP 备案（国内服务器）

### 2. API 调用限制
| 账号类型 | 日调用量 | QPS |
|----------|----------|-----|
| 个人测试 | 100 次/天 | 5 次/秒 |
| 企业账号 | 10000 次/天 | 50 次/秒 |
| 高级合作 | 不限 | 协商 |

### 3. 审核要点
- 应用描述要清晰
- 回调地址要可访问
- 隐私政策要完善
- 不能直接复制 1688 商品（需要加工）

```
应用名称：即时零售比价工具

应用描述：
本应用面向电商卖家和采购商，提供智能比价服务。
用户授权登录后，应用会：
1. 获取用户的历史订单信息
2. 在 1688 平台搜索同款商品
3. 对比价格并推荐更优选择

- 电商卖家采购比价
- 个人用户购物省钱

- 使用 OAuth 2.0 安全授权
- 仅获取必要信息（用户信息、订单信息）
- 数据本地加密存储
- 不保存用户敏感信息

隐私保护：
- 用户可随时取消授权
- 支持数据导出和删除
- 遵守 GDPR 和网络安全法
```
