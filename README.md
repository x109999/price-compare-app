# 即时零售比价工具

一个现代化的即时零售比价 Web 应用，帮助用户自动抓取历史订单并在 1688 上寻找同款低价商品。

## 功能特点

- 🎯 **欢迎页面** - 清晰展示产品功能和价值
- 📱 **扫码登录** - 支持淘宝/1688 APP 扫码授权
- 🔄 **实时进度** - 可视化展示比价进度（抓取订单 → 1688 比价 → 生成报告）
- 💰 **比价结果** - 智能展示节省金额和推荐商品
- 📊 **数据导出** - 支持导出比价报告

## 页面流程

```
欢迎页 → 扫码登录 → 比价进行中 → 比价结果
```

## 主要功能说明

### 1. 欢迎页面
- 展示应用名称和核心价值
- 三大功能亮点展示
- 引导用户开始使用

### 2. 扫码登录
- 动态生成二维码（模拟）
- 3 秒后自动模拟扫码成功
- 支持刷新二维码

### 3. 比价进行中
- 三步进度展示：抓取订单 → 1688 比价 → 生成报告
- 实时进度条和百分比
- 动态统计数据（订单数、预计节省）

### 4. 比价结果
- 总体节省统计卡片
- 推荐商品列表
- 支持查看详情
- 导出报告功能

## 模拟数据说明

当前版本使用模拟数据演示完整流程：

- 优惠幅度 20-50%
