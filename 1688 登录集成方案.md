# 1688 真实扫码登录集成方案

## 方案说明

要实现真实的 1688 扫码登录，需要以下步骤：

---

## 🔑 前置准备

### 1. 申请 1688 开放平台账号

1. 访问：https://open.1688.com/
2. 注册开发者账号
3. 创建应用，获取：
   - `app_key` (应用密钥)
   - `app_secret` (应用密钥)
   - `redirect_uri` (回调地址)

### 2. 配置应用权限

- 申请 `user_info` 权限（获取用户信息）
- 申请 `order_info` 权限（获取订单信息，用于比价）

---

## 🚀 实现方案

### 方案 A：使用后端服务（推荐）

**架构**：
```
前端 → 你的后端 → 1688 API
```

**步骤**：

1. **后端生成二维码**
   - 调用 1688 OAuth 接口获取授权 URL
   - 生成二维码展示给用户

2. **用户扫码授权**
   - 用户用 1688 APP 扫码
   - 1688 回调你的后端

3. **获取用户信息**
   - 用授权码换取 access_token
   - 调用用户信息 API

**后端代码示例（Node.js）**：

```javascript
const express = require('express');
const axios = require('axios');
const crypto = require('crypto');

const app = express();
const APP_KEY = 'your_app_key';
const APP_SECRET = 'your_app_secret';
const REDIRECT_URI = 'http://yourdomain.com/callback';

// 生成授权二维码
app.get('/api/1688/qr', (req, res) => {
    const state = crypto.randomBytes(16).toString('hex');
    const authUrl = `https://oauth.1688.com/oauth2/authorize?client_id=${APP_KEY}&redirect_uri=${encodeURIComponent(REDIRECT_URI)}&response_type=code&state=${state}`;
    
    // 生成二维码
    const qrCode = `https://api.qrserver.com/v1/create-qr-code/?data=${encodeURIComponent(authUrl)}`;
    
    res.json({ qrCode, state });
});

// 回调处理
app.get('/callback', async (req, res) => {
    const { code, state } = req.query;
    
    // 换取 access_token
    const tokenRes = await axios.post('https://oauth.1688.com/oauth2/token', {
        client_id: APP_KEY,
        client_secret: APP_SECRET,
        grant_type: 'authorization_code',
        code: code,
        redirect_uri: REDIRECT_URI
    });
    
    const { access_token, expires_in } = tokenRes.data;
    
    // 获取用户信息
    const userRes = await axios.get('https://open.1688.com/api/user/info', {
        headers: { Authorization: `Bearer ${access_token}` }
    });
    
    res.json({ success: true, user: userRes.data });
});

app.listen(3000);
```

## 🔧 前端修改

将模拟登录改为调用真实 API：

```javascript
// 生成真实二维码
async generateQRCode() {
    const qrContainer = document.getElementById('qr-code');
    qrContainer.innerHTML = '<div class="spinner"></div>';
    
    try {
        const res = await fetch('/api/1688/qr');
        const { qrCode, state } = await res.json();
        
        qrContainer.innerHTML = `<img src="${qrCode}" alt="扫码登录">`;
        
        // 轮询检查登录状态
        this.pollLoginStatus(state);
    } catch (error) {
        qrContainer.innerHTML = '<p>二维码生成失败</p>';
    }
}

// 轮询登录状态
async pollLoginStatus(state) {
    const interval = setInterval(async () => {
        const res = await fetch(`/api/1688/check?state=${state}`);
        const data = await res.json();
        
        if (data.loggedIn) {
            clearInterval(interval);
            this.user = data.user;
            this.showPage('processing-page');
        }
    }, 2000);
}
```

## ⚠️ 注意事项

1. **HTTPS 要求**：1688 OAuth 要求回调地址必须是 HTTPS
2. **域名备案**：如果使用国内服务器，域名需要备案
3. **审核周期**：1688 应用创建后需要审核（1-3 个工作日）
4. **API 配额**：免费账号有调用次数限制
