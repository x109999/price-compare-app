使用阿里云轻量应用服务器（SAS）或 ECS，国内访问速度极快（50ms 以内）。

## 🔧 部署步骤

### 步骤 1：购买服务器（5 分钟）

1. **注册/登录阿里云**
   - 访问：https://www.aliyun.com/
   - 注册账号（需要手机号）
   - 实名认证（必需）

2. **选择轻量应用服务器**
   - 进入：https://swas.console.aliyun.com/
   - 点击"创建服务器"
   - 选择配置：
     - 地域：选择离您近的（北京/上海/深圳）
     - 镜像：Ubuntu 20.04
     - 套餐：¥99/月（2 核 2G 5M）
   - 点击"立即购买"

3. **设置密码**
   - 设置 root 密码（记住！）
   - 确认购买

### 步骤 2：连接服务器（2 分钟）

**方式 A：网页终端（最简单）**
1. 进入控制台：https://swas.console.aliyun.com/
2. 点击服务器卡片
3. 点击"远程连接"
4. 输入 root 密码登录

**方式 B：SSH 连接**
```bash
# 替换为您的服务器 IP
ssh root@your_server_ip
# 输入密码
```

### 步骤 3：一键部署（5 分钟）

**在服务器上执行**：

```bash
# 1. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs git nginx

# 2. 克隆代码
cd /root
git clone https://github.com/x109999/price-compare-app.git
cd price-compare-app

# 3. 安装依赖
cd backend
npm install --production
cd ..

# 4. 运行部署脚本
chmod +x deploy.sh
./deploy.sh
```

### 步骤 4：配置域名（可选）

**如果有域名**：

1. **域名备案**（必需）
   - 访问：https://beian.aliyun.com/
   - 提交资料（1-3 个工作日）

2. **DNS 解析**
   - 登录域名管理后台
   - 添加 A 记录：
     - 主机记录：`www` 或 `@`
     - 记录值：服务器 IP
     - TTL：10 分钟

3. **配置 HTTPS**
   ```bash
   # 在服务器上执行
   sudo apt-get install certbot python3-certbot-nginx
   sudo certbot --nginx -d yourdomain.com
   ```

## 🔐 配置 1688 登录

```bash

```

## 🆘 常见问题

### Q: 服务器连不上？
A: 检查安全组规则，确保 80/443/22 端口开放

### Q: 访问超时？
A: 检查 Nginx 是否运行：`sudo systemctl status nginx`

### Q: 后端服务挂了？
A: 重启服务：`sudo systemctl restart price-compare.service`

### Q: 如何查看日志？
A: `sudo journalctl -u price-compare.service -f`
