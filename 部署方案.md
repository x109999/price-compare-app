## 🎯 方案一：Vercel 部署（最简单 ⭐推荐）

### 优势
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN
- ✅ 自动部署
- ✅ 无需服务器

### 步骤

#### 1. 准备 GitHub 账号
如果没有 GitHub 账号，访问 https://github.com 注册

#### 2. 创建 GitHub 仓库
```bash
cd /home/admin/openclaw/workspace/price-compare-app
git init
git add .
git commit -m "Initial commit"
```

#### 3. 推送到 GitHub
```bash
# 替换为你的 GitHub 用户名和仓库名
git remote add origin https://github.com/YOUR_USERNAME/price-compare-app.git
git branch -M main
git push -u origin main
```

#### 4. 部署到 Vercel

**方式 A：使用 Vercel 网页**
1. 访问 https://vercel.com/new
2. 点击 "Import Git Repository"
3. 选择刚才创建的仓库
4. 点击 "Deploy"
5. 等待部署完成（约 1 分钟）

**方式 B：使用 Vercel CLI**
```bash
# 安装 Vercel
npm install -g vercel

# 登录
vercel login

# 部署
cd /home/admin/openclaw/workspace/price-compare-app
vercel --prod
```

#### 5. 配置环境变量
1. 进入 Vercel 项目页面
2. Settings → Environment Variables
3. 添加：
   - `APP_KEY`: 你的 1688 app_key
   - `APP_SECRET`: 你的 1688 app_secret
   - `REDIRECT_URI`: https://your-app.vercel.app/callback

#### 6. 获取网址
部署完成后，您会获得：
```
https://price-compare-app-xxx.vercel.app
```

## ☁️ 方案二：阿里云服务器部署

### 步骤

#### 1. 购买服务器
- 访问 https://www.aliyun.com/
- 选择 ECS 云服务器
- 配置推荐：
  - CPU: 1 核
  - 内存：2GB
  - 带宽：3Mbps
  - 系统：Ubuntu 20.04
- 价格：约 ¥99/月（新用户优惠）

#### 2. 连接服务器
```bash
# 替换为你的服务器 IP
ssh root@your_server_ip
```

#### 3. 上传代码
```bash
# 在本地执行
cd /home/admin/openclaw/workspace
tar -czf price-compare-app.tar.gz price-compare-app/
scp price-compare-app.tar.gz root@your_server_ip:/root/
```

#### 4. 一键部署
```bash
# 在服务器上执行
cd /root
tar -xzf price-compare-app.tar.gz
cd price-compare-app
chmod +x deploy.sh
./deploy.sh
```

#### 5. 配置域名（可选）
1. 购买域名（约 ¥60/年）
2. 域名备案（阿里云代备案）
3. DNS 解析到服务器 IP
4. 运行 `./deploy.sh` 配置 HTTPS

#### 6. 获取网址
```
http://你的服务器 IP
或
https://你的域名
```

## 🔧 配置 1688 登录

无论选择哪个方案，都需要：

### 1. 申请 1688 开放平台
参考 `1688 开放平台申请指南.md`

### 2. 配置密钥
```bash
# 编辑 .env 文件
nano backend/.env

# 填入
APP_KEY=你的 app_key
APP_SECRET=你的 app_secret
REDIRECT_URI=你的回调地址
```

### 3. 重启服务
```bash
# Vercel：自动重启
# 服务器部署
sudo systemctl restart price-compare.service

# 本地部署
npm start
```
